Para escribir un documento de latex necesitaremos los siguientes componentes:

\begin{enumerate}
  \item part
  \item chapter
  \item section
  \item subsection
  \item enumerate
  \item itemize
  \item theo
  \item cor
  \item lem
  \item defn
  \item prop
  \item obs
  \item dem
  \item ejm
  \item nota
  \item cons
\end{enumerate}
